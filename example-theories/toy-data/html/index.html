theory-7 Name: Hybrid Structured and Episodic Memory Enables Robust, Generalizable LLM Agent Performance in Text Games
Description: Language model agents for text-based games achieve superior robustness, sample efficiency, and generalization when augmented with hybrid memory systems that combine explicit structured (e.g., knowledge-graph or belief-graph) state representations and episodic (trajectory or reflection-based) memory. Structured memory provides compact, relational representations that support multi-hop reasoning, navigation, and action-space pruning, while episodic memory enables recall of detailed instructions, procedures, and successful/failed experiences. The synergistic integration and principled retrieval of both kinds of memories enables agents to overcome both partial observability and long-horizon credit assignment bottlenecks, outperforming agents with either memory alone, with unstructured memory (e.g., full text history), or with naive retrieval/summarization.

theory-6 Name: Compression and Selection Mechanisms Are Essential to Overcome Scaling Limits for In-Context Memory in LLM Text-Game Agents
Description: This theory posits that as text-game environments scale in observation length, object count, and task horizon, LLM and RL-based agents must rely on memory compression (e.g., diff-history, summarization, area selection) and relevance-based selection (e.g., retrieval, recency/importance scoring) rather than full retention of episode histories in prompt context. Without these, agents hit hard resource and sample efficiency ceilings due to context-window length, token costs, and cognitive overload. Compression must not discard critical state changes; selection must ensure salient, task-relevant cues are retained.

These principles, when systematically implemented, enable agents to surpass the limitations of naive context concatenation or indiscriminate storage, yielding significant gains in sample efficiency, planning horizon, and generalization to novel tasks.

theory-1 Name: Hybrid Structured-Episodic Memory as a Fundamental Requirement for Robust Performance in LLM-Based Text Game Agents
Description: This theory posits that robust performance, generalization, and strategic flexibility in language model (LLM) agents across complex, partially observable text games require not just large memory but a principled integration of structured (e.g., knowledge/belief graph) and episodic/reflection-based memory. The synergy emerges because structured memory offers compact relational representations for navigation and reasoning, while episodic/reflection memory provides dynamic adaptation, procedural recall, and learning from both failure and success. Neither naive retention of all history nor pure reliance on unstructured episodic memory alone is sufficient for scaling or generalizing to new, difficult, or long-horizon tasks; selective retrieval and composition of these modalities is necessary to balance information compression, retrieval relevance, and depth of reasoning.
