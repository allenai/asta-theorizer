Theory (General Information)

ID: theory-2

Name: Information Compression and History Representation are Key Limiting Factors in LLM Agent Memory Utility

Type: general

Theory Query: Build a theory of how language model agents can best be augmented with memory to perform well at text games, based on the following results.

Description: This theory asserts that in text-based game environments with large, high-dimensional or verbose observations, the utility of memory for LLM agents is fundamentally constrained by the information-theoretic compressibility (i.e., sufficient representation) of experience. Explicitly, compressive forms of context/history (e.g., graph-diff, Unix diff, structured summaries) vastly outperform equally-sized or even much larger uncompressed or full-history approaches, by ensuring that relevant signals outlast window constraints and guide agent policy effectively. Token budget, compression ratio, and choice of memory summarization directly determine effective planning horizon and strategic depth.

Theory (Statement/Laws)

Theory Statements/Laws

Statement 0: Memory Compression Ratio Determines Planning Depth and Performance

Statement: The effective planning and recall horizon of LLM-based agents in text games is quantitatively bounded by the compression ratio of the memory representation: Given a fixed token/context size, agents with higher compression modes (e.g., diff, graph-diff, structured area/summary) can represent deeper and more salient past experience in context, yielding higher sample efficiency, faster convergence, and better asymptotic performance than full-history or verbose episodic memory agents.

Domain/Scope: Applies to any LLM-based agent in settings with high-dimensional, verbose, or rapidly growing history (e.g., NetHack, complex IF, games with combinatorial state), especially when context/prompt size is a bottleneck.

Special Cases

  1. In environments with inherently low-dimensional state (e.g., small grids), the benefits of compression are diminished.
  2. If relevant information is single-step and always present, compression gives marginal value.

Supporting Evidence for this Law

Theory (Additional Details)

New Predictions (Likely outcome)

  1. For any given prompt/context size, memory strategies yielding greater average compression (relative to original observations/action history) will enable successful task performance on longer-horizon instances than those using uncompressed/raw history.
  2. In high-dimensional or verbose-text games (e.g., NetHack, Zork), agents with graph-diff, area-selection, or other compressive memory will scale to deeper horizons without drop-off, while full-history or verbose episodic agents will fail or saturate at shallow steps.
  3. Adding an auxiliary prediction objective (e.g., world-model or diff-prediction) to compressed memory potentiates even faster convergence, but is less critical for already highly compressed histories.

New Predictions (Unknown outcome/high-entropy)

  1. A hybrid strategy that adaptively selects among multiple compressive encodings (diff, graph-diff, summary) as a function of environment dynamics will further extend effective horizon or generalize to games with mixed text/state density.
  2. In emergent narrative or creative IF domains, novel learned compression (beyond current heuristics) will enable cross-episode or even meta-memory composition for persistent recurring characters/goals.
  3. Compression-based memory agents will, at sufficient scale, discover strategies invisible to full-history or reflection-only agents in games featuring deceptive observations or high-stakes planning.

Negative Experiments

  1. If a non-compressive, verbose history agent achieves equal or better performance (sample efficiency and asymptotic score) at equal context/prompt size in a high-dimensional/complex text game, the theory would be falsified.
  2. If compression degrades planning (e.g., induces information loss that cannot be remedied by additional tokens) across all environments, undermining the law's predictions.
  3. If auxiliary world-model objectives only harm or never improve compressed-memory agents, it would refute the auxiliary benefit predicted.

Unaccounted for Evidence

  1. In babyAI-style environments, even though diff increases average token count, it still improves sample efficiency, suggesting additional factors like alignment with causal structure may also be in play. [e18.0]
