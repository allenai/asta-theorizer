Predictive Accuracy Evaluation for Theory theory-1

Theory Statement Being Evaluated

Theory ID: theory-1

Theory Statement: The benefit of memory augmentation to LLM agents in text games depends not just on the presence or volume of stored experience, but crucially on how memory is structured (graphs, attributes, summaries), accessed (contextual semantic retrieval, attribute filtering, embedding similarity, graph traversal), and composed/interleaved with ongoing reasoning. Unstructured, brute-force context window or naive concatenation yields information overload or sparsity and fails to scale in complex games.

Number of Papers Evaluated: 36

Number of Rubric Elements with Some Evidence: 4

Specific Prediction: LLM agents using structured memory representations (e.g., knowledge graphs, belief graphs, attribute-indexed retrieval) outperform agents using unstructured memory (e.g., naive concatenation of full history or raw episodic logs) on complex text games with non-trivial state spaces and partial observability.
